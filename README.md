# 🎮 Raycast Pygame Renderer

[![Language: Russian](https://shields.io)](README.ru.md)

A lightweight, fast, and optimized pseudo-3D raycaster engine built using Python, Pygame, and NumPy, featuring advanced post-processing visual effects.

### ✨ Features
* **Post-Processing:** Dynamic god rays and lens focus/blur effects on Left Mouse Button hold.
* **Particle System:** Real-time sparks rendering for interactive "welding" effects.
* **Textured Environment:** Fully textured walls, floors, and ceilings.
* **Modern Controls:** Smooth Mouse Look + WASD keyboard layout.
* **High Performance:** Math calculations accelerated with NumPy vectorization.

### 🚀 Quick Start

#### Requirements
* Python 3.9+
* Pygame
* NumPy

#### Installation & Launch
1. Clone the repository:
   ```bash
   git clone https://github.com
   cd py_raycast_project
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run:
   ```bash
   python run.py
   ```

### 🎮 Controls
* **WASD** — Movement
* **Mouse** — Look around
* **Left Mouse Button (LMB)** — Focus / Blur effect + Weld sparks
