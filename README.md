# 🎮 Raycast Pygame Renderer

Простой и быстрый рейкастер на Python + Pygame с эффектами пост-обработки:
- 🔦 God rays (лучи света)
- 🎯 Фокус/размытие при зажатой ЛКМ
- ✨ Система частиц (искры при "сварке")
- 🧱 Текстурированные стены, пол и потолок
- 🖱️ Управление мышью + WASD

![Превью](assets/preview.png)

## 🚀 Быстрый старт

### Требования
- Python 3.9+
- [Pygame](https://www.pygame.org/)
- [NumPy](https://numpy.org/)

### Установка
```bash
git clone https://github.com/yourusername/raycast-pygame.git
cd raycast-pygame
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
