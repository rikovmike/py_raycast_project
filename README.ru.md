# 🎮 Рейкаст Рендерер на Pygame

[![Language: English]](README.md)

Простой, быстрый и оптимизированный псевдотрехмерный движок рейкастинга на Python, Pygame и NumPy с продвинутыми эффектами пост-обработки.

### ✨ Особенности
* **Пост-обработка:** Динамические лучи света (God rays) и эффект фокуса/размытия при зажатии левой кнопки мыши.
* **Система частиц:** Реалистичные искры в реальном времени (эффект "сварки").
* **Текстурирование:** Полностью текстурированные стены, пол и потолок.
* **Современное управление:** Плавный обзор мышью + клавиатурная раскладка WASD.
* **Высокая производительность:** Математические расчеты ускорены за счет векторизации в NumPy.

### 🚀 Быстрый старт

#### Требования
* Python 3.9+
* Pygame
* NumPy

#### Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/rikovmike/py_raycast_project.git
   cd py_raycast_project
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите проект:
   ```bash
   python run.py
   ```

### 🎮 Управление
* **WASD** — Перемещение
* **Мышь** — Обзор во все стороны
* **Левая кнопка мыши (ЛКМ)** — Эффект фокуса/размытия + Искры сварки
